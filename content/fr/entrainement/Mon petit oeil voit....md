## Description
Lire les possibilités de mouvement dans l'environnement

## Organisation
⌛Durée: 10min

👥Participants: groupes de 3-7 participants

⚙️Matériel: obstacles divers

## Description
Un participant annonce un mouvement à réaliser, par exemple: "mon petit oeil voit... un tic-tac !". Tous les autres doivent aller réaliser le plus rapidement possible ce mouvement et revenir à la zone de départ. Le dernier arrivé devient celui qui annonce le prochain mouvement.

## Variantes
➖Au lieu d'un nom de mouvement, l'annonceur dit le nom d'un obstacle ou type de surface. Les autres peuvent utiliser n'importe quelle technique de parkour sur cet obstacle ou surface.

➕L'annonceur donne un enchainement (2-5 mouvements), par exemple "mon petit oeil voit... un tic-tac préc', saut de chat-bras !"

## Notes
Les participants doivent avoir en tête le nom d'au moins cinq ou six techniques pour que le jeu fonctionne.





Une personne dit un mvmt à réaliser. Les joueurs doivent réaliser le mvmt puis revenir à la zone de départ. Le dernier arrivé a perdu (donner un gage ?)
