## Description
Une main est placée au sommet de l'obstacle, l'autre sur la surface verticale opposée afin d'effectuer une rotation pour passer les jambes par-dessus l'obstacle.

## Remarques
Ce passement est particulièrement utile pour les obstacles hauts, puisqu'il permet d'avoir le centre de gravité en-dessous du sommet de l'obstacle avant de quitter l'appui.
